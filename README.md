# Bank Loan Report – Task 3: Dashboard & Deep Analysis

## 📌 Project Overview
This task focuses on building an interactive Tableau dashboard and performing deep-dive analysis on bank loan data.

---

## 📊 Dashboard Features
- KPI tracking (Total Loans, Good/Bad Loans)
- Monthly trends
- State-wise analysis
- Purpose-based insights
- Interactive filters

---

## 🔍 Deep Dive Analysis
- Loan Status Analysis (Good vs Bad Loans)
- State-wise loan distribution
- Loan purpose trends

---

## 🛠️ Tools Used
- Tableau
- SQL
- Data Analysis techniques

---

## 📁 Files Included
- Tableau Workbook (.twbx)
- Dashboard screenshots
- Deep dive report
- KPI definitions

---

## 📈 Key Insights
- Majority loans are good loans
- Certain states dominate loan applications
- Loan purpose trends reveal customer behavior

---

## 🚀 Conclusion
The interactive dashboard provides a comprehensive view of loan data, enabling better business decision-making.
